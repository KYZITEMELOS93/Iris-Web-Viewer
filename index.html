    Iris Web Viewer Chrome Extention [v 0.1 - Beta]

    Iris Web Viewer is a cross platform 3D Model Viewer for exchange formats.

    It allows for a number of different model formats to be viewed as well as exposes the raw data of the model such as Vertice/Normal data, UV Texture Coordinates among other things, without the need for expensive and large 3D modeling systems.

    Supported Formats
    • *.stl
    • *.obj
