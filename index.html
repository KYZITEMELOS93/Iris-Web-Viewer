    Iris Web Viewer - About

    Iris Web Viewer is a cross platform 3D Model Viewer for exchange formats.

    Supported Formats


    The entire point of IRIS is to allow for a central location to view 3D file formats. If there's a format you want added, contact us below and we'll do our best to add it in.

    Currently IRIS supports:
    - *.stl (both ASCII and binary)
    - *.obj

    Contact and Follow


    Problem, question, concern? IRIS is opensource and hosted on Github, check the issues section or post a question there. If you want to reach me directly you can tweet to me on twitter.
